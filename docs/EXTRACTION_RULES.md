# Extraction Rules

The template is a skeleton, not a product clone.

## Extract

- Short `AGENTS.md` maps.
- Stable architecture boundaries.
- Context documents that help agents resume work.
- Mechanical quality gates.
- Trellis-compatible task and spec guidance.
- Upgrade manifests and preview-only update flow.

## Do Not Extract

- Domain entities from one source project.
- Platform adapters, account flows, or import models.
- Historical handoffs, screenshots, smoke evidence, or local data.
- Old task boards from source projects.
- Private paths, user identity, credentials, or environment-specific state.

## Promotion Rule

Do not promote a business conclusion into the template unless it has been validated across at least two unrelated projects.
